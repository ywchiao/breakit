# 前言

 前一章嘗試了基本的 [canvas][mdnCanvas2D] 2D 繪圖，這一章開始
 進入**動畫** (*animation*) 的世界。

## 專案準備

 和前一章比較，這一章開始將接觸到 [JavaScript][mdnJavaScript]
 的**物件** (*object*) 系統；[JavaScript][mdnJavaScript] 的物件系統
 和 *C++/C#/Java* 等語言的物件系統非常不同；所以是個全新的開始。
