# 前言

 之前介紹了 [JavaScript][mdnJavaScript] 的 *literal objects* (用在
  ```ball``` 物件)；這一章，將進入 [JavaScript][mdnJavaScript] 的
  class*。

 這一章將利用 *class* 將**擊球板** ```paddle``` 和**磚塊** ```brick``` 
 都轉換成**物件**；同時，為了遊戲進行方便，也加上了**開始**，**暫停**，
 等控制按鈕。
